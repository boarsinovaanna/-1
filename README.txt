 Угадай число
Простая консольная игра на C++. Компьютер загадывает случайное число от 1 до 100, а игрок пытается его угадать.
 Установка и запуск
 Требования
- Компилятор C++ (например, g++, clang++)
 Компиляция
Откройте терминал в папке с файлом `guess_number.cpp` и выполните:
    bash
g++ guess_number.cpp -o guess_number
Запуск
· Linux / macOS:
    bash
  ./guess_number
· Windows:
    bash
  guess_number.exe
Правила игры
· После запуска программа загадывает число.
· Вводите целые числа в консоль.
· Каждая попытка сопровождается подсказкой: «Больше!» или «Меньше!».
· Когда число угадано, программа показывает количество попыток.
· Можно сыграть несколько раундов подряд.
Особенности реализации
· Весь исходный код находится в одном файле (guess_number.cpp).
· Присутствует защита от некорректного ввода (буквы, символы).
· Генерация случайных чисел через rand() с инициализацией srand(time(0)).
